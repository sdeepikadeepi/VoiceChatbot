# 🎤 Voice Chatbot

An interactive Voice Chatbot developed using HTML, CSS, JavaScript, and Web Speech API.

## 🚀 Features

- Voice input using speech recognition
- Text-to-speech spoken replies
- Female voice response
- Emoji chat messages
- Date and time stamps
- Clear chat button
- Attractive colorful UI
- Smooth animations

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Web Speech API
- Git & GitHub

## 📂 Project Files

- index.html
- style.css
- script.js

## ▶ How to Run

1. Download project files
2. Open in VS Code
3. Run with Live Server
4. Click Speak button
5. Allow microphone permission

## 📌 Project Purpose

This project allows users to interact with a chatbot using voice commands and receive spoken responses in a modern chat interface.
